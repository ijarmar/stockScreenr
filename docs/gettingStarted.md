# Getting started

### API - https://api.iextrading.com/1.0/

#### Start by importing StockData.php

```PHP
    require_once('yourPathToLibrary/StockData.php');
```


#### Create a class reference

```PHP
    $screenr = new StockData(string $apiURL);
```
